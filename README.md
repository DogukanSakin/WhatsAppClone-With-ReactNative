# WhatsApp UI Clone with React Native
This repository contains only the basic part of the UI of the WhatsApp application. Completely dummy data is used in the application. Dummy data (such as message data) only exists for some dummy users. The UI designs of the application are based only on the version currently available on the Play Store. Designs may differ in the App Store.  

## Installation

Clone this repository on your local machine.

```
git clone https://github.com/DogukanSakin/WhatsAppClone-With-ReactNative.git
```


## Usage

Run the following commands in the project folder to install the project dependencies.

```
npm install
npx expo start
```
